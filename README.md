# Dotfiles

My personal dotfiles.

## Installation

Install the following applications.

- `stow`
- `neovim`
- `tmux`
- `lazygit`

You can run the `install.sh`.

```bash
sudo chmod +x ./install.sh
./install.sh
```

## Setup

Setup the dotfiles by running `setup.sh`.

```bash
sudo chmod +x ./setup.sh
./setup.sh
```
